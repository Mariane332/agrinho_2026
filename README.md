# agrinho_2026
O preojeto Agro Sustenta foi um projeto criado em sala de aula. 
Pelo projeto Agrinho Programação, usando ferramentas de inteligência artificial para a construção do site.
O Objetivo foi mostrar "Agro forte, futuro sustentável: equilíbrio entre produção e meio ambiente" é um tema de extrema importância.
Toda a parte do HTML foi de autoria própria,focando no tema disponibilizado pelo programa Agrinho.
Para geração das imagens foi usada a ferramenta Canva IA e apenas uma delas encontrada no google.
Para disign e criação de animações foi usado auxilio do Chat GPT para encontsr códigos que nunca foram usados por mim antes.
Contando com um questionário simples parar treinar seus conhecimentos adquiridos no site.
Contudo, tanto o HTML< CSS e o JS foram feitos com todo cuidado para se tornar o mais confortável possível a navegação pelo site.
Obrigado. 💚
